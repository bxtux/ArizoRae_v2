# Sites emploi par métier

Cette fiche regroupe quelques canaux spécialisés par grande famille,
indépendamment du pays. Elle n'est ni exhaustive, ni taxonomique : le
skill étant agnostique au métier, elle sert de point de repère rapide
si `preset.md` est indisponible. Dès que `preset.md` (construit par
recherche web à l'init) contient une section "Jobboards et canaux",
cette dernière prime.

Combiner avec le fichier pays correspondant pour composer une liste
utilisable sur un marché donné.
