Skill ArizoRAE

Bundle Cowork — à installer via Claude Code CLI

↓ Télécharger rae-generic.skill
1

Installer Claude Code

Claude Code est le CLI officiel Anthropic. Il nécessite Node.js 18+.

npm install -g @anthropic-ai/claude-code
2

Installer les dépendances Python

Nécessaires pour extraire le texte des CV (PDF, DOCX, ODT).

pip install pymupdf python-docx striprtf
3

Installer le skill

Téléchargez rae-generic.skill ci-dessus, puis :

# Depuis le dossier où vous avez téléchargé le fichier
claude skill install ./rae-generic.skill

Ou depuis le registre Cowork :

claude skill install rae-generic
4

Initialiser votre profil

Ouvrez Claude Code dans votre répertoire de travail, puis lancez la commande d'init. Prévoyez votre CV (PDF, DOCX ou TXT) et 10-15 minutes. 

# Ouvrir Claude Code
claude

# Dans Claude Code, taper :
/init

Claude Code vous guidera : chemin du CV, métier visé, pays, questions de complétion. À la fin, editez FACTS.md — c'est la seule étape manuelle obligatoire.

5

Utiliser le skill

Toujours depuis Claude Code :

# Analyser une offre (coller le texte ou donner un fichier)
/analyse <offre>

# Générer un CV adapté
/cv <offre>

# Rédiger la lettre de motivation
/lettre <offre>

# Préparer un entretien
/entretien <offre>
⌨

Terminal web

Claude Code est pré-installé avec le skill rae-generic. Authentifiez-vous avec votre compte Claude et lancez /init directement dans le navigateur.

Ouvrir le terminal →
Compte requis : gratuit ou payant sur claude.ai ou une clé API Anthropic (ANTHROPIC_API_KEY).

Confidentialité : votre session est isolée et effacée à la déconnexion. Aucun fichier n'est conservé côté serveur.
Étapes dans le terminal :
1. Exécuter claude → suivre le lien OAuth ou entrer votre API key
2. Taper /init dans Claude Code
3. Télécharger vos fichiers avec download ~/workspace/FACTS.md
